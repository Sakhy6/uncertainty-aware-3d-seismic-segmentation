# Uncertainty-Aware 3D Seismic Facies Segmentation Using Deep Learning

## Overview

This repository contains the implementation of my Master's research on uncertainty-aware 3D seismic facies segmentation using deep learning. The project introduces an uncertainty estimation framework based on predictive entropy to improve the reliability and interpretability of segmentation results.

The research was presented at the IEEE ICCE-TW 2026 International Conference and completed as part of my Master's degree in Electrical and Communication Engineering at Yuan Ze University.

## Features

- 3D Residual Dilated U-Net
- Predictive entropy uncertainty estimation
- Risk map generation
- 3D seismic facies segmentation
- Deep learning using PyTorch

## Technologies

- SciPy

- OpenCV

## Results

The proposed method achieves highly accurate 3D seismic facies segmentation while providing uncertainty maps that help identify regions where model predictions are less reliable.
